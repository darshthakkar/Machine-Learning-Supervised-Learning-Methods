How to run the code

1. Open cmd/terminal.
2. Navigate into the 'code' folder.
3. Type in cmd/terminal 'python <filename>.py'. Example: 'python SVM.py'

Note: The system needs to have required packages to run the code. sci-kit, keras, tensorflow, numpy, pandas etc
	  All the graphs that will be generated when the code is run can be saved by the user manually if required.
